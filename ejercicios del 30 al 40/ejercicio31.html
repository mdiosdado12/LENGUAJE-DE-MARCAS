    • Introducción

    SQL, Structure Query Language (Lenguaje de Consulta Estructurado), es un lenguaje de programación para trabajar con bases de datos relacionales como MySQL, Oracle, etc.
    MySQL es un interpretador de SQL, es un servidor de base de datos.

    show databases

    Una base de datos es un conjunto de tablas.
